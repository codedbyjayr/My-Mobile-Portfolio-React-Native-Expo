# My-Mobile-Portfolio-React-Native-Expo
A native mobile application built to showcase my professional journey, technical projects, and web system expertise. Designed with a focus on high-performance UI and seamless navigation, this app serves as a "living resume" accessible directly on Android and iOS via Expo Go.
🚀 Features
Interactive Project Gallery: A curated list of my work (including web systems and hardware simulations) with deep-links to GitHub repositories.

Dynamic Skill Matrix: A visual breakdown of my proficiency in Web Technologies, Server Administration, and Embedded Systems.

Real-time Contact Form: Integrated messaging to connect with me instantly.

Education & Experience Timeline: A clean, vertical timeline tracking my academic progress and key milestones.

Optimized Dark Mode: A sleek, developer-centric aesthetic designed for modern mobile displays.

🛠️ Tech Stack
Framework: React Native

Workflow: Expo Go (Managed Workflow)

Navigation: React Navigation (Stack & Tab)

Icons: Expo Vector Icons (Lucide/Material Design)

Animations: React Native Reanimated for fluid transitions.

📦 Mobile-Specific Implementation
Since this is a native app, I focused on:

Flexbox Layouts: Ensuring the UI scales perfectly across different screen aspect ratios.

Native Feedback: Using Pressable and TouchableOpacity to provide tactile feedback common in mobile UX.

Expo Constants: Handling status bar heights and device-specific safe areas.
